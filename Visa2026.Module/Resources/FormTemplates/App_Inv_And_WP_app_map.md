Generated by `tools/GenerateTemplates` → `MakeAppInvAndWPLetterTemplate`. Merge model prefix `ds` matches `AppInvAndWPLetterReportDef` (same logical fields as the XAF report, sourced from **`Application`** and related `ProjectContract`, `Company`, `CompanyHead`, etc.).

**Placeholder parity:** Body sentences use the same merge keys as the map / `AppInvAndWPReport` (`[Company.Name]` → `Company_Name`, `[TotalPersonCount]` / `[TotalPersonCountText]`, `[VisaPeriod_NameTm]`, `[VisaCategory_NameTm]`, attachments expression).

**Word typography vs XAF RTF (`App_Inv_And_WP_app.jpg`):**

| Region | Word template (scan) | XAF `xrRichBody2` (map `**…**`) |
|--------|------------------------|----------------------------------|
| `ProjectContract_Description` (Maksady body) | Justified, plain, **no** first-line indent | `xrRichBody1`: justified, `\fi720` |
| `[Company.Name]` in request paragraph | **Bold** | Plain in map excerpt |
| `[TotalPersonCount] ([TotalPersonCountText])`, `[VisaPeriod_NameTm] möhlet`, `[VisaCategory_NameTm] çakylyk we iş rugsatnamasyny` | Plain | **Bold** in map |
| Ministry recipient block | Right-aligned, bold | Label position differs (report designer) |

Salutation: bold + underlined (scan). Urgency: italic, underlined, red (scan; XAF italic). Attachments: same strings as `xrLabelAttachments` expression.
