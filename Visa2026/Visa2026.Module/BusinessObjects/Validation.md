# Business Object Validation Rules

This document describes validation rules applied to properties in various Business Objects.

## Application

| Property Name | Validation Rule | Description |
|---|---|---|
| `ApplicationDate` | `RuleRequiredField` | Ensures that the Application Date is not empty. |
| `OrganizationType` | `RuleRequiredField` | Ensures that the Organization Type is selected. |
| `ApplicationType` | `RuleRequiredField` | Ensures that the Application Type is selected. |
